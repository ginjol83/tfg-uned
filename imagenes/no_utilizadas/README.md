# Imágenes no utilizadas

Esta carpeta contiene imágenes que están disponibles en el proyecto pero que no se referencian actualmente en ningún archivo .tex del documento.

## Contenido:

### Directorio principal (`no_utilizadas/`):
- `Boton_Nuevo.png` - Botón de nuevo (no referenciado)
- `Boton_Pegar.png` - Botón de pegar (no referenciado) 
- `Boton_Rehacer.png` - Botón de rehacer (no referenciado)
- `CapturaMenuIzq.png` - Captura de menú izquierdo (no referenciado)
- `Principal_con_errores.png` - Pantalla principal con errores (redundante con Pantalla_Principal.png)
- `Principal_con_warnings.png` - Pantalla principal con warnings (redundante con Pantalla_Principal.png)

### Subcarpeta `botones_manual/`:
Contiene 14 imágenes de botones del manual que no se utilizan actualmente:
- Diversos botones de la interfaz (Abrir, Cerrar, Compilar, Guardar, etc.)

### Subcarpeta `Vistas/`:
Contiene 17 imágenes de la interfaz del programa que no se referencian en el documento:
- Pantallas de diálogos de configuración
- Editores de tiles
- Vistas de consola y monitor  
- Pantalla principal y variantes

## Nota:
Estas imágenes se mantienen aquí por si en el futuro se necesitan para el documento o para versiones futuras del manual.
