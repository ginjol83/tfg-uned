\chapter*{Resumen} % El * indica que no queremos que añada la palabra "Capitulo"

Este trabajo aborda el desarrollo de un entorno de desarrollo integrado (IDE) enfocado al desarrollo de software en C para máquinas de 8 bits. El objetivo principal es que la aplicación soporte el desarrollo para múltiples plataformas de ordenadores y videoconsolas de las décadas de 1980 y 1990 basadas en tecnología de 8 bits. Para ello, se ha desarrollado un software utilizando principalmente tecnologías Java, capaz de compilar código mediante distintos compiladores dirigidos a estas arquitecturas y que, además, incorpora un sistema de plugins que permite ampliar y extender sus funcionalidades.

Los resultados muestran que la herramienta proporciona un entorno unificado que simplifica el flujo de trabajo en el desarrollo para arquitecturas de 8 bits, mejorando la accesibilidad, la organización y la eficiencia del proceso.

\chapter*{Abstract} % El * indica que no queremos que añada la palabra "Capitulo"

This work addresses the development of an Integrated Development Environment (IDE) focused on C software development for 8-bit systems. The main objective is to enable the application to support development for multiple platforms of computers and video game consoles from the 1980s and 1990s based on 8-bit technology. To this end, a software solution has been developed primarily using Java technologies, capable of compiling code through different compilers targeting these architectures and incorporating a plugin system that allows the extension and enhancement of its functionalities.

The results show that the tool provides a unified environment that simplifies the workflow in 8-bit architecture development, improving accessibility, organization, and overall process efficiency.
